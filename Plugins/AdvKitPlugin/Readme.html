Adventure Kit

Thank you for purchasing the Adventure Kit!

The Adventure Kit is a plugin for the Unreal Engine 4 that includes gameplay elements which can be used as a basis for a variety of games.

License

Use of this software is permitted in accordance with Unreal Engine Marketplace guidelines. If it was purchased elsewhere here is the run down: By purchasing this software it is licensed to you for the lifetime right to use the content in developing an unlimited number of products and in shipping those products. You are also licensed to make the content available to employees and contractors for the sole purpose of contributing to products controlled by you. Each such product built using content purchased must be a standalone creative work (such a game, simulation, or video) in which content is merely a component and not the primary focus.

GitHub Access

To get access to the GitHub repository please forward the purchase receipt email to githubaccess@freetimestudio.net and include your GitHub username in the form "username: YourGitUserName" (not your registered email). While you are allowed to share this software with your team only one GitHub user is allowed per purchase.
